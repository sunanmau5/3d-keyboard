# 3D Keyboard Visualization

Real-time 3D keyboard visualization built with Three.js. Auto-detects your keyboard layout and animates key presses as you type.

## Features

- Real-time 3D rendering with smooth animations
- Auto-detects keyboard layout (ANSI, ISO, German, UK)
- Manual layout switching
- 60 FPS animations with dynamic lighting
- Zero build tools - pure JavaScript

## Quick Start

```bash
# serve locally (required for ES6 modules)
npm run dev

# or use Python
python3 -m http.server 8000
```

Open `http://localhost:8000` and start typing!

## Browser Support

Modern browsers with WebGL support (Chrome, Firefox, Safari, Edge)
